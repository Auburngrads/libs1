hot_context_menu {rhandsontable}R Documentation

Handsontable widget

Description

Configure the options for the right-click context menu. See Context Menu and jquery contextMenu for details.

Usage

hot_context_menu(hot, allowRowEdit = TRUE, allowColEdit = TRUE,
  allowReadOnly = FALSE, allowComments = FALSE,
  allowCustomBorders = FALSE, customOpts = NULL, ...)

Arguments

hot

rhandsontable object

allowRowEdit

logical enabling row editing

allowColEdit

logical enabling column editing. Note that Handsontable does not support column add/remove when column types are defined (i.e. useTypes == TRUE in rhandsontable).

allowReadOnly

logical enabling read-only toggle

allowComments

logical enabling comments

allowCustomBorders

logical enabling custom borders

customOpts

list

...

ignored

Examples

library(rhandsontable)
DF = data.frame(val = 1:10, bool = TRUE, big = LETTERS[1:10],
                small = letters[1:10],
                dt = seq(from = Sys.Date(), by = "days", length.out = 10),
                stringsAsFactors = FALSE)

rhandsontable(DF) %>%
  hot_context_menu(allowRowEdit = FALSE, allowColEdit = FALSE)
