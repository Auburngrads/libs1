hot_cell {rhandsontable}R Documentation

Handsontable widget

Description

Configure single cell. See Handsontable.js for details.

Usage

hot_cell(hot, row, col, comment = NULL)

Arguments

hot

rhandsontable object

row

numeric row index

col

numeric column index

comment

character comment to add to cell

See Also

hot_cols, hot_rows

Examples

library(rhandsontable)
DF = data.frame(val = 1:10, bool = TRUE, big = LETTERS[1:10],
                small = letters[1:10],
                dt = seq(from = Sys.Date(), by = "days", length.out = 10),
                stringsAsFactors = FALSE)

rhandsontable(DF, readOnly = TRUE) %>%
  hot_cell(1, 1, "Test comment")
