plot.graph {BDgraph}R Documentation

Plot function for S3 class "graph"

Description

Visualizes structure of the graph.

Usage

## S3 method for class 'graph'
plot( x, main = NULL, layout = layout.circle, ... )

Arguments

x

An object of S3 class "graph", from function graph.sim.

main

Graphical parameter (see plot).

layout

The vertex placement algorithm which is according to igraph package.

...

System reserved (no specific usage).
